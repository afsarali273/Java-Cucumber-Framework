Quick Start

1. Clone the Repository

git clone https://github.com/YOUR_USERNAME/SandsAutoFramework.git
cd SandsAutoFramework

2. Build the Project

mvn clean install

3. Run Tests

# Run all tests
mvn clean test

# Run UI tests only
mvn test -Dcucumber.filter.tags="@UI"

# Run API tests only
mvn test -Dcucumber.filter.tags="@API"

4. View Reports

# Open HTML report
open test-output/reports/CustomReport_*.html

# Generate Allure report
mvn allure:serve

System Requirements

  • Java: JDK 11 or higher
  • Maven: 3.6+ for build management
  • Browsers: Chrome, Firefox, Edge, Safari (for web testing)
  • Node.js: Optional, for Playwright browser installation
  • Appium: For mobile automation (Android/iOS)
  • WinAppDriver: For Windows desktop automation
  • OS: Windows, macOS, Linux
