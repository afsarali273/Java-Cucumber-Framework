PlaywrightReusable - All Methods

Complete reference of all methods available in PlaywrightReusable class.

Usage: Extend PlaywrightReusable in your Page Object classes to access all these methods.

Playwright-Specific Examples

Using Locator Methods Pattern

public class LoginPage extends PlaywrightReusable {
    // Locator methods - always fresh
    private Locator usernameField() { return locator("#username"); }
    private Locator passwordField() { return locator("#password"); }
    private Locator loginButton() { return locator("#loginBtn"); }
    
    public void login(String username, String password) {
        type(usernameField(), username);
        type(passwordField(), password);
        click(loginButton());
        
        // Use native assertion
        assertVisible(locator(".dashboard"));
    }
}

Using getBy Methods

// By role
click(getByRole(AriaRole.BUTTON, new Page.GetByRoleOptions().setName("Submit")));

// By text
click(getByText("Login"));

// By placeholder
type(getByPlaceholder("Enter email"), "test@example.com");

// By label
type(getByLabel("Password"), "password123");

// By test ID
click(getByTestId("submit-button"));

Native Assertions

// Element state
assertVisible(locator("#result"));
assertEnabled(locator("#submitBtn"));
assertChecked(locator("#agreeTerms"));

// Text
assertContainsText(locator(".message"), "Success");
assertHasText(locator("h1"), "Welcome");

// Attributes
assertHasAttribute(locator("a"), "href", "/home");
assertHasClass(locator("div"), "active");

// Page
assertPageHasTitle("Dashboard");
assertPageURLContains("/dashboard");

Custom Assertions

// Using expect() for custom assertions
expect(locator("#price")).hasValue("$99.99");
expect(locator(".error")).not().isVisible();
expect(page).hasURL("https://example.com/home");
