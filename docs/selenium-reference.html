Selenium Complete Reference

SeleniumReusable - All Methods

Complete reference of all methods available in SeleniumReusable class.

Usage: Extend SeleniumReusable in your Page Object classes to access all these methods.

Navigation Methods

Method Parameters Description
navigateTo() String url Navigate to specified URL
navigateBack() - Navigate back in browser history
navigateForward() - Navigate forward in browser history
refreshPage() - Refresh current page

Element Interaction Methods

Method Parameters Description
click() By locator Click on element
type() By locator, String text Clear and type text into element
typeAndEnter() By locator, String text Type text and press Enter
clear() By locator Clear element text
getText() By locator Get element text content
getAttribute() By locator, String attribute Get element attribute value
getCssValue() By locator, String property Get CSS property value
jsClick() By locator Click using JavaScript
doubleClick() By locator Double click on element
mouseHover() By locator Hover mouse over element
dragAndDrop() By source, By target Drag source to target

Dropdown Methods

Method Parameters Description
selectDropdownByText() By locator, String text Select option by visible text
selectDropdownByValue() By locator, String value Select option by value attribute
selectDropdownByIndex() By locator, int index Select option by index (0-based)

Checkbox & Radio Methods

Method Parameters Description
check() By locator Check checkbox if not already checked
uncheck() By locator Uncheck checkbox if checked
isChecked() By locator Returns true if checkbox is checked

Wait Methods

Method Parameters Description
waitForElementToBeVisible() By locator Wait until element is visible
waitForElementToBeClickable() By locator Wait until element is clickable
waitForElementToBePresent() By locator Wait until element is present in DOM
waitForElementToBeInvisible() By locator Wait until element becomes invisible
waitForAttributeToContain() By locator, String attribute, String value Wait until attribute contains value
waitForSeconds() int seconds Hard wait for specified seconds

Scroll Methods

Method Parameters Description
scrollToElement() By locator Scroll to bring element into view
scrollToTop() - Scroll to top of page
scrollToBottom() - Scroll to bottom of page

Window/Tab Management

Method Parameters Description
switchToWindow() String windowHandle Switch to window by handle
closeCurrentWindow() - Close current window/tab
openNewTab() - Open new browser tab
switchToLatestTab() - Switch to most recently opened tab

Screenshot Methods

Method Parameters Description
takeScreenshot() String filePath Capture full page screenshot
takeElementScreenshot() By locator, String filePath Capture screenshot of specific element

Selenium 4 Relative Locators

Method Parameters Description
findElementAbove() By locator, By aboveLocator Find element above reference element
findElementBelow() By locator, By belowLocator Find element below reference element
findElementNear() By locator, By nearLocator Find element near reference element
findElementToLeftOf() By locator, By leftLocator Find element to left of reference
findElementToRightOf() By locator, By rightLocator Find element to right of reference

Utility Methods

Method Parameters Description
isDisplayed() By locator Check if element is displayed
isEnabled() By locator Check if element is enabled
findElements() By locator Find all matching elements
uploadFile() By locator, String filePath Upload file to input element
getElementRect() By locator Get element rectangle (x, y, width, height)
getShadowRoot() By locator Access Shadow DOM root element

Assertion Methods

Method Parameters Description
assertEquals() Object actual, Object expected, String message Assert two values are equal
assertNotEquals() Object actual, Object expected, String message Assert two values are not equal
assertTrue() boolean condition, String message Assert condition is true
assertFalse() boolean condition, String message Assert condition is false
assertNull() Object object, String message Assert object is null
assertNotNull() Object object, String message Assert object is not null
assertContains() String actual, String expected, String message Assert string contains substring
enableSoftAssert() - Enable soft assertions mode
assertAll() - Verify all soft assertions
